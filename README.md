# GenExprForSublime

##What's that?
It's a group of snippets for developing gen code for [Max](http://www.cycling74.com).




##Installation
go into your *~/Library/Application Support/Sublime Text 3/Packages/User* folder and drop the **GenExprForSublime** folder in it.
